Teknisk info

Følgende info er utdatert, men vil bli oppdatert etter hvert.

Innlogging

  • Innlogging: Bruker/passord mottatt ved innsjekking. Domene IDIPC.
  • Brukerområde: Z:
  • Starte PC^2: Z:\PC2\pc2team.bat
  • Pålogging PC2: teamN/teamN (for passende verdier av N)

Textpad

  • Starte Textpad: Start -> All Programs -> Textpad
  • Fikse '}': Configure -> Preferences -> Keyboard. Fjern ALT-0 fra View + ActiveClipLib.

Emacs

  • Starte Emacs: Start -> All Programs -> GNU Emacs -> . (Problemer med { og }: Kan lages med CTRL-ALT-7 og CTRL-ALT-0).

VI

  • Kan lastes ned fra www.vim.org og installeres på Z:, hvis du heller vil bruke dette enn Emacs og Textpad.

Kjøring fra kommandolinjen

  • Java kommandolinje: Start -> Run -> Cmd, javac Klasse.java, java Klasse < innput.txt
  • C/C++ kommandolinje: Start -> All programs -> Microsoft Visual Studio .NET 2003 -> Visual Studio .NET Tools -> Visual Studio .NET Command Prompt, cl /EHsc foo.c, foo.exe < innput.txt

Visual Studio programmeringsmiljø

  • Visual Studio programmeringsmiljø: Start -> All programs -> Microsoft Visual Studio .NET 2003 -> Microsoft Visual Studio .NET 2003

Utskrift

Du kan skrive ut programmene dine på skriveren \\printhost\p336 hvis du vil debugge på papir. Be en dommer hente utskriften.

Toalett

Finnes innerst i gangen.

Test

Du kan teste at det tekniske oppsettet fungerer på følgende måte:

  • Lag katalogen Z:\test
  • Lagre Test.java, test.c, test.cpp og test.in på Z:\test (listet under)
  • Start C/C++ kommandolinjeskall (se over)
  • Z:
  • cd \test
  • javac Test.java
  • java Test < test.in
  • cl test.c
  • test.exe < test.in
  • cl /EHsc test.cpp
  • test.exe < test.in

Disse programmene skal skrive ut: 5.86.

Test.java

import java.io.*;
import java.text.*;
import java.util.*;
public class Test {
    static DecimalFormat DF = 
            new DecimalFormat("0.00", new DecimalFormatSymbols(Locale.ENGLISH));
    public static void main(String[] args) throws Exception {
        BufferedReader bf = new BufferedReader(new InputStreamReader(System.in));
        StringTokenizer st = new StringTokenizer(bf.readLine());
        double a = Double.parseDouble(st.nextToken());
        double b = Double.parseDouble(st.nextToken());
        System.out.println(DF.format(a + b));
    }
}
