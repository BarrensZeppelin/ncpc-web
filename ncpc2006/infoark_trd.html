Teknisk info

Innlogging

Merk at det er forskjellig brukernavn og passord for innlogging i Windows og PC^2

  • Innlogging: Bruker/passord mottatt ved innsjekking. Domene IDIPC.
  • Brukerområde: Z:\work
  • Starte PC^2: Z:\pc2v87\pc2team.bat
  • Pålogging PC2: teamN/teamN (for passende verdier av N)

Textpad

  • Starte Textpad: Start -> All Programs -> Textpad
  • Fikse '}': Configure -> Preferences -> Keyboard. Fjern ALT-0 fra View + ActiveClipLib.

Emacs

  • Starte Emacs: Start -> All Programs -> GNU Emacs -> . (Problemer med { og }: Kan lages med CTRL-ALT-7 og CTRL-ALT-0).

VI

  • Kan lastes ned fra www.vim.org og installeres på Z:, hvis du heller vil bruke dette enn Emacs og Textpad.

Kompilering og testing av programmer

Du kan teste at det tekniske oppsettet fungerer på følgende måte:

Java

  • Lag katalogen Z:\work
  • Java påkommandolinjen:
    • Lagre Test.java og test.in på Z:\work (listet under)
    • Start -> Run -> Cmd
    • Z:
    • cd work
    • javac Test.java
    • java Test < test.in
    • Sjekk at svaret ble riktig.
  • Eclipse
    • Start -> All programs -> Eclipse 3.2
    • File -> Switch workspace... -> Workspace: Z:\work
    • File -> New -> Project -> Java Project -> Enter project name: NCPC2006 -> Finish
    • File -> New -> File -> File name: test.in -> Finish
    • File -> New -> Class -> Enter name: Test -> Finish
    • Run -> Run... -> Arguments -> Program arguments: src/test.in -> Run
    • Sjekk at svaret ble riktig.

C/C++

  • Lag katalogen Z:\work
  • C/C++ på kommandolinjen:
    • Lagre test.c/test.cpp og test.in på Z:\work (listet under)
    • Start -> All programs -> Microsoft Visual Studio 2005 -> Visual Studio Tools -> Visual Studio 2005 Command Prompt
    • Z:
    • cd work
    • cl /EHsc test.c eller cl /EHsc test.cpp
    • test.exe < test.in
    • Sjekk at svaret ble riktig.

Test.java

import java.io.*;
import java.util.*;

public class Test {

    public static void main(String[] args) throws Exception {
        Locale.setDefault(Locale.UK);
        if (args.length > 0) {
            // read from file ...
            br = new BufferedReader(new FileReader(args[0]));
        }
        else {
            // ... or read from standard in
            br = new BufferedReader(new InputStreamReader(System.in));
        }
        double a = readDouble();
        double b = readDouble();
        System.out.printf("%.2f\n", a + b);
    }

    static BufferedReader br;
    static StringTokenizer st = new StringTokenizer("");

    static String readString() throws Exception {
        while (!st.hasMoreTokens()) {
            st = new StringTokenizer(br.readLine());
        }
        return st.nextToken();
    }

    static int readInt() throws Exception {
        return Integer.parseInt(readString());
    }

    static double readDouble() throws Exception {
        return Double.parseDouble(readString());
    }
}
